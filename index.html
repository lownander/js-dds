“Hover” spell

Oh look, it's Carola!.

During the finale at Carola's most recent concert, Harry Potter has promised to make her levitate.

But he's fed up of pressing buttons. He wants it so when you hover your mouse over the button, Carola will levitate. When you take your mouse off the button (called "hovering off") she should stop levitating.

Exercise 5: Shopping Cartz

 Your cart (0 items)

Wow! This site is a store!? What can I buy!? IKEA HOTDOGS*

  1. There is a function in the javascript file called addToCart. Make it so when you click the 'Add to cart' button, it calls that function. You know you got it right if in your javascript console, it says added an item! each time you press the button.
  2. Modify the addToCart function so that when you press the button, the fades out and in. This gets the user's attention and shows them that something was added.
  3. Modify the addToCart function so that when you press the button, the number of items in the cart is updated. The variable numItems represents the number of items in the cart. The html for the cart needs to be updated with this number.
